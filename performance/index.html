Performance comparison of Javascript Parsing Generators/Libraries

Each of the following tools was used to implement a JSON Parser:

  1. Jison parser Generator grammar generated code

  2. Chevrotain Parsing DSL implementation

  3. Antlr4 Parser Generator grammar generated code

  4. PEG.js Parser Generator grammar generated code

  5. Parsimmon Parser Combinator implementation

  6. Nearley Parser Generator grammar generated code hand built lexer

  7. Ohm-js Parser Library grammar parser

The input sample for the benchmark is a ~1,000 lines JSON file generated by json-generator.

All the parsers have no output(build no AST), they only parse the text (without errors)

Benchmarking is done using the benchmark.js library.

The entire code for this benchmark can be found in this github repo.

Please be patient as this benchmark can take 1-2 minutes.
